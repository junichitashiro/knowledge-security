# マルウェア

## 語源
- Malicious（悪意のある）
- Software
- → 組み合わせた造語

## 分類

### トロイの木馬
- コンピュータに潜伏
- 情報の窃取
- 攻撃の踏み台として利用

### ウィルス
- 狭義のコンピュータウィルス
- 他ファイルに寄生して感染拡大

### ワーム
- 独自に感染
- 自己増殖（寄生なし）

### ランサムウェア
- データを暗号化
- 復号と引き換えに金銭を要求

### スケアウェア
- 偽の警告で恐怖心を煽る
- 金銭・情報の提供を要求
